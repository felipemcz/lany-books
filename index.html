⚙️ Configurações
Gerencie seus dados e preferências ✦
Exportar dados
Salva todos os seus livros, conquistas, moedas e configurações num arquivo .json. Guarde esse arquivo para restaurar em outro navegador ou dispositivo.

💡 Dica: exporte sempre que terminar uma sessão de leitura importante. O arquivo fica salvo na pasta de Downloads do seu dispositivo.

Importar dados
Selecione um arquivo .json exportado anteriormente para restaurar seu progresso. Atenção: substituirá todos os dados atuais.

💡 Como mudar de navegador:
1. No navegador atual → clique em Baixar backup
2. No novo navegador → abra o app e vá em Configurações
3. Clique em Selecionar arquivo .json e escolha o backup baixado
4. Pronto! Todo o progresso será restaurado ✅
